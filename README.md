# Edge Typing Bot
A Selenium-based bot for automated typing tests.

## Setup
1. Install Python 3.12.
2. Run `pip install selenium`.
3. Download the [Edge Driver](https://developer.microsoft.com/en-us/microsoft-edge/tools/driver/) that matches your browser version.
4. Place `msedgedriver.exe` in the root folder.
5. Run `python final_bot.py`.
